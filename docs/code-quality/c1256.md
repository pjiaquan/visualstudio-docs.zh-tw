# C1256
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C1256: PCH 必須同時被寫入和讀取。  
  
 當工具中有內部錯誤而不是在分析程式碼時，程式碼分析工具會回報這個警告。
