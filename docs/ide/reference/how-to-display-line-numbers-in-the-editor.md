# 如何：顯示編輯器中的行號
<a id="how-to-display-line-numbers-in-the-editor" class="xliff"></a>
您可以在程式碼中顯示或隱藏行號。  
  
> [!NOTE]
>  根據您目前使用的設定，出現的對話方塊與功能表命令可能不同於本文所述。 若要變更設定，請開啟 [工具]/[匯入和匯出設定]。 如需詳細資訊，請參閱[將 Visual Studio IDE 個人化](../../ide/personalizing-the-visual-studio-ide.md)。  
  
## 顯示程式碼行號
<a id="display-line-numbers-in-code" class="xliff"></a>  
 行號並未加入至程式碼中，只是當做參考而已。 若要列印行號，請在 [列印] 對話方塊中選取 [包含行號] 核取方塊。  
  
#### 若要顯示程式碼行號
<a id="to-display-line-numbers-in-code" class="xliff"></a>  
  
1.  在功能表列上選擇 [工具] 、[選項] 。 展開 [文字編輯器] 節點，然後選取所使用語言的節點，或選取 [所有語言] 開啟所有語言的行號。 也可以在 [快速啟動] 方塊中輸入**行號**。  
  
2.  ![在編輯器中顯示行號的選項](~/docs/ide/reference/media/vs_displaylinenumbers.png "VS_DisplayLineNumbers")  
  
> [!NOTE]
>  根據您的語言或設定，您可能需要選取 [選項] 對話方塊中的 [顯示所有設定] 核取方塊，才能顯示 [所有語言] 子節點。
