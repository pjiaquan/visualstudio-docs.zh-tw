# 偵錯工具視窗
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

您可以在偵錯程式時開啟大部分偵錯工具視窗。 若要查看偵錯工具視窗的清單，請設定中斷點並開始偵錯。 當您叫用中斷點且執行停止時，按一下 \[偵錯\] \/ \[Windows\]。
