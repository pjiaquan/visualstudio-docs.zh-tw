# 個別執行緒摘要報表
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

此橫條圖會顯示目前可見的時間範圍內，每個未隱藏執行緒在每個活動類別所花費的時間。 「執行」表示執行緒正在執行；所有其他類別則表示執行緒正在等待某個項目。
