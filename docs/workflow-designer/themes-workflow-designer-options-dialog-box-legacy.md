# 選項對話方塊、工作流程設計工具、佈景主題 (舊版)
本主題描述如何在舊版 [!INCLUDE[wfd1](../workflow-designer/includes/wfd1_md.md)] 中使用 \[**佈景主題、Workflow 設計工具、選項**\] 對話方塊。當您需要以 [!INCLUDE[netfx35_long](../workflow-designer/includes/netfx35_long_md.md)] 或 [!INCLUDE[vstecwinfx](../workflow-designer/includes/vstecwinfx_md.md)] 為目標時，請使用舊版 [!INCLUDE[wfd2](../workflow-designer/includes/wfd2_md.md)]。  
  
 佈景主題會定義工作流程的背景色彩、前景色彩、樣式、圖示和其他視覺化項目。您可以儲存佈景主題以供其他工作流程重複使用。您可以使用 \[**佈景主題、工作流程設計工具、選項**\] 對話方塊，選取預設佈景主題和已儲存的自訂佈景主題。若要開啟這個對話方塊，請按一下 \[**工作流程**\] 功能表上的 \[**選取自訂主題**\]。或者，以滑鼠右鍵按一下工作流程設計介面，並按一下 \[**選取自訂主題**\]。  
  
 下表說明 \[**佈景主題、工作流程設計工具、選項**\] 對話方塊的使用者介面 \(UI\) 項目。  
  
|UI 項目|描述|  
|-----------|--------|  
|**新增**|按一下以開啟[主題組態對話方塊 \(舊版\)](../workflow-designer/theme-configuration-dialog-box-legacy.md)，建立新的佈景主題。|  
|**修改...**|按一下以開啟[主題組態對話方塊 \(舊版\)](../workflow-designer/theme-configuration-dialog-box-legacy.md)，修改選取的佈景主題。|  
|**瀏覽**|按一下以瀏覽工作流程主題檔案，其副檔名為 .wtm。|  
|**選取主題**|主題清單的主題名稱資料行標題。選取清單中的主題，按一下 \[**修改**\] 可編輯，或按一下 \[**確定**\] 當做工作流程的主題使用。|  
|**說明**|主題清單的主題描述資料行標題。|  
|**位置**|主題清單的主題位置資料行標題。|
