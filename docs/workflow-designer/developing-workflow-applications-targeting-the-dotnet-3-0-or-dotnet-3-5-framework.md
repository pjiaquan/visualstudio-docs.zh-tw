# 以 .NET 3.0 或 .NET 3.5 Framework 為目標開發工作流程應用程式
[!INCLUDE[vs2010](../modeling/includes/vs2010_md.md)] 會提供 [!INCLUDE[wfd1](../workflow-designer/includes/wfd1_md.md)] 來建立以 [!INCLUDE[netfx40_long](../workflow-designer/includes/netfx40_long_md.md)] 為目標的 [!INCLUDE[wf](../workflow-designer/includes/wf_md.md)] 應用程式。設計工具會使用熟悉的 [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] 使用者介面進行工作流程應用程式的圖形建構與偵錯。  
  
 [!INCLUDE[vs2010](../modeling/includes/vs2010_md.md)] 也會提供舊版工作流程設計工具的存取，該設計工具可用來以 [!INCLUDE[netfx35_long](../workflow-designer/includes/netfx35_long_md.md)] 或 [!INCLUDE[vstecwinfx](../workflow-designer/includes/vstecwinfx_md.md)] 為目標。本章節包含使用這個舊版設計工具的指引。
