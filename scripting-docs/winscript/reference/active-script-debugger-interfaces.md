# 動態指令碼偵錯工具的介面
activdbg.h 和 activdbg100.h 標頭檔提供一節中列出的介面、列舉型別和結構。  它們是偵錯指令碼。  
  
> [!NOTE]
>  `IJSDebug*` 介面和 `IEnumJsStackFrames` 介面在 Internet Explorer 11 先釋放了偵錯機器碼的與指令碼。  這些介面的標頭檔是 jscript9diag.h。
